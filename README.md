# Simple API in Go with Fiber and Docker


## Env file
Create .env file using the .env.example file
 
## Docker
### Run Docker
Clone this repository and up project run:
```
docker-compose up
```
or
```
docker compose up
```
### Edit code
After making changes in the code run:
```
docker run build
```
